How to Create a GitHub Webpage

Step 1 Create a github account

Step 2 In the repositories page, select New

Step 3 Enter a repository name and click "Create Repository"

Step 4 Under "Quick Setup", select "Uploading an Existing File."

Step 5 Create an Index.html file on your computer using a text editor or code editor

Step 6 Select "choose your files" and select the newly created Index.html file

Step 7 Select "Commit CHanges"

Step 8 Select "Settings" and then navigate to "Pages" under the "Code and autiomation" section

Step 9 Under "Branch" select "main" and click "Save"

Step 10 Navigate back to the "Code" section and click "github-pages" under Environments

Step 11 Select "View Deployment"
